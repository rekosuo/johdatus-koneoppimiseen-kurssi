# Johdanto

**Reko Suonsivu**

Minulla on koneoppimiseen liittyen kokemusta Stable Diffusionin hienosäätämisestä, erityisesti LoRA-malleja kouluttamalla. En kuitenkaan ole syventänyt tietoani niin paljon, että todella ymmärtäisin Stable Diffusioniin tai sen hienosäätöön sisältyvän logiikan.

Datan käsittelystä minulla on kokemusta kurssilta *Datan käsittelyn kehittyneet menetelmät*, ja olen muutenkin kokenut Pythonin käyttäjä.

Ennakko-odoksia tältä kurssilta ei ole muita, kuin että opettajan edellisien kurssien laadusta päätellen tämäkin tulee olemaan laadukas kurssi.
